# Gemini Sheet Builder

Desktop приложение на Tkinter для генерации изображений через Gemini с помощью промта и опциональных шаблонных изображений (фон, стиль, шрифт, пример, персонаж). Поддерживает несколько листов, повторную генерацию, предпросмотр и сбор всех понравившихся изображений в PDF, который по умолчанию сохраняется на рабочий стол.

## Возможности
- Несколько листов с разными промтами и наборами шаблонов.
- Выбор соотношения сторон и разрешения (1K/2K/4K).
- Прикрепление до пяти вспомогательных изображений (фон, стиль, шрифт, пример, персонаж).
- Повторная генерация для листа, если результат не понравился.
- Автосохранение всех сгенерированных изображений и метаданных локально в папке `output`.
- Сбор всех текущих изображений в один PDF (по умолчанию `~/Desktop/gemini_output.pdf`).
- Кроссплатформенность: проверено для macOS и Windows (зависит от наличия Python + Tkinter).

## Установка зависимостей
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
```

## Запуск
```bash
python -m src.app
```

Перед запуском задайте API-ключ в переменной окружения `GEMINI_API_KEY` или введите его прямо в поле интерфейса.

## Использование
1. Введите промт, выберите соотношение сторон и разрешение.
2. При необходимости добавьте изображения-шаблоны через кнопки "Choose".
3. Нажмите **Generate** для получения изображения. Файл и метаданные сохранятся в `output/`.
4. Добавляйте новые листы кнопкой **Add Sheet** и переключайтесь между ними в списке слева.
5. Если результат не нравится, нажмите **Regenerate** на выбранном листе.
6. Когда все изображения готовы, нажмите **Save All to PDF**, чтобы объединить их в один файл (по умолчанию на рабочий стол).

## Замечания
- Все изображения и JSON-метаданные сохраняются локально в каталоге `output` независимо от экспорта в PDF.
- Для корректной работы на macOS/Windows требуется установленный Python 3.9+ с поддержкой Tkinter.
